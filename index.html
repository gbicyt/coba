Kenapa gratis, untungnya dari mana?

Kami non-profit tujuan utama kami untuk membantu masyarakat yang terbentur biaya dalam mengakses layanan cloud services. Selain itu semua software yang kami gunakan juga open source jadi kami tidak memerlukan biaya lisensi aplikasi. Jika anda ingin membantu project kami anda dapat memberikan sumbangan melalui menu donasi di atas atau dalam bentuk apapun termasuk server dalam kondisi bekas atau baru (part server, server fisik, hardisk , ssd, colocation, switch, router, IP public, dll)
